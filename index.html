Назначение

Cетевой интерфейс PRESP32

Сетевой интерфейс presp32 предназначен для расширения коммуникационных возможностей программируемых реле серии ПР200, ПР102, ПР100. В зависимости от используемой прошивки и модификации плат, доступны следующие сервисы и протоколы:

  • Modbus TCP
  • Modbus RTU
  • Telegram Bot
  • Home Kit
  • MQTT
  • 1-wire(ds18b20)
  • Logger
  • Digital-IO
  • OTA

Интерфейс WI-FI/Ethernet

Подключение к сети через WI-Fi в режиме станции STA и точки доступа AP. В разработке версия с подключением к сети через Ethernet, с возможностью использовать WI-FI в качестве резервного канала связи.

Telegram Bot

Поддержка чтения/записи регистров через Telegram Bot. Получение информации по настройкам и сервисам.

Home Kit

Реализовано на выбор 4 профиля. Bridge и Single accessory для управление освещением до 24 дискретных каналов, возможность управлять выходами реле при помощи голосовых команд. Temperature sensor до 8 каналов, для ввода данных с регистров Modbus в приложение Дом. Termostat до 4 каналов, для создания интерфейсов термостата в разных режимах. Быстрая интеграция через QR-Code.

Логгер

Поддержка логирования на ПК через script файл и на microSD карту. Возможность выбирать разные форматы для архвирования: uint16, uint32, float, bin. Скачивание файлов с sd карты по сети через встроенный файловый менеджер без извлечения карты из платы. Для регистрации быстрых процессов запись с периодом 100мс, 500мс. Для синхронизации времени используется внешний NTP сервер.

Настройка

Все настройки осуществляются через встроенный WEB интерфейс. настройки сохраняются в энергонезависимой памяти. Обновление прошивки без физического доступа к плате через OTA. Для контроля работы и состояния интерфейса на плате установлен светодиод, его режимы можно настроить через WEB интерфейс.

Протоколы и интерфейсы

Поддерживаются проводные и беспроводные протоколы для доступа к регистрам. Для связи с программируемыми реле используется Modbus RTU. Modbus TCP, MQTT используется для создания визуализации на телефоне, ПК. Telegram Bot для быстрого доступа к информации с минимальными настройками.

Модификации интерфейсных плат

Модификации интерфейсных плат

Существует несколько модификаций сетевых интерфейсов для подключения к программируемым реле. Встариваемый вариант для установки в модификацию ПР200, поддержан режим Master и Slave. Обмен данными через UART по протоколу Modbus RTU. Для подключения внешних датчиков 1-WIRE/I2C возможна установка 4 pin разъема. Поддержка microSD карты для записи архивов.

Вариант внешнего интерфейса, подключается через RS485, можно использовать совместно с модификациями ПР200, ПР102, ПР100, подключение через WI-FI. В случае использования внешних подключений по 1-wire/I2C рекомендуется использовать внешний вариант платы, в этом случае осуществляется гальваническая развязка через сетевой интерфейс ПР. Поддержка microSD карты для записи архивов.

Вариант в виде USB стика, можно использовать так же как внешний вариант, или как отладочный комплект при пусконаладочных работах на объекте, подключение через WI-FI. Поддержка microSD карты для записи архивов.

Версия с подключением по Ethernet. Подходит если в месте установки связь с WI-FI неустойчивая. Функционал аналогичен предыдущим версиям, только связь через Ethernet.
