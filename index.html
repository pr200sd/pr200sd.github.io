PRESP32

Сетевой интерфейс presp32 предназначен для расширения коммуникационных возможностей программируемых реле серии ПР200, ПР102, ПР100. В зависимости от используемой прошивки и модификации плат, доступны следующие сервисы и протоколы:

  • modbus TCP
  • Modbus RTU
  • Telegram Bot
  • HomeKit
  • MQTT
  • 1-wire(ds18b20)
  • Logger
  • Digital-IO

Существует несколько модификаций сетевых интерфейсов для подключения к программируемым реле.


Вариант для установки в модификацию ПР200, подключение через WI-FI


Вариант внешнего интерфейса, подключается через RS485, можно использовать совместно с модификациями ПР200, ПР102, ПР100, подключение через WI-FI


Вариант в виде USB стика, можно использовать так же как внешний вариант, или как отладочный комплект при пусконаладочных работах на объекте, подключение через WI-FI


Версия с подключением по Ethernet
