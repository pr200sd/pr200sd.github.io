сетевой интерфейс PRESP32

Сетевой интерфейс presp32 предназначен для расширения коммуникационных возможностей программируемых реле серии ПР200, ПР102, ПР100. В зависимости от используемой прошивки и модификации плат, доступны следующие сервисы и протоколы:

  • Modbus TCP
  • Modbus RTU
  • Telegram Bot
  • Home Kit
  • MQTT
  • 1-wire(ds18b20)
  • Logger
  • Digital-IO
  • OTA

Интерфейс WI-FI/Ethernet

Подключение к сети через WI-Fi в режиме станции STA и точки доступа AP. В разработке версия с подключением к сети через Ethernet, с возможностью использовать WI-FI в качестве резервного канала связи.

Telegram Bot

Поддержка чтения/записи регистров через Telegram Bot

Home Kit

Поддержка сервиса Home Kit для управление освещением до 24 дискретных каналов, возможность управлять выходами реле при помощи голосовых команд

Логгер

Поддержка логирования на ПК через json файл и на microSD карту. Скачивание файлов с sd карты по сети через встроенный файловый менеджер без необходимости доставать карту из платы.

Настройка

Все настройки осуществляются через ыстроенный WEB интерфейс. настройки сохраняются в энергонезависимой памяти. Обновление прошивки без физического доступа к плате через OTA.

Протоколы и интерфейсы

Поддерживаются проводные и безпроводные протоколы для связи. Для связи с программируемыми реле используется Modbus RTU. Modbus TCP, MQTT используется для создания визуализации на телефоне, ПК.

Модификации интерфейсных плат

Существует несколько модификаций сетевых интерфейсов для подключения к программируемым реле. Встариваемый вариант для установки в модификацию ПР200, поддержан режим Master и Slave. Обмен данными через UART по протоколу Modbus RTU.

Для подключения внешних датчиков 1-WIRE/I2C возможна установка 4 pin разъема.
